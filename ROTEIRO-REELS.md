## COMO GRAVAR

### Opcao 1: Gravador de Tela do Windows (Recomendado)
1. Abra `reels-animacao.html` no Chrome
2. Pressione **F** para tela cheia (ou clique "Fullscreen")
3. Pressione **H** para esconder os botoes Replay/Fullscreen
4. Use **Win + Shift + R** (Ferramenta de Recorte) ou **Win + G** (Xbox Game Bar) para gravar
5. Pressione **R** para reiniciar a animacao e comece a gravar imediatamente
6. A animacao dura 20 segundos — pare a gravacao quando a barra vermelha chegar ao fim

### Opcao 2: Celular
1. Abra `reels-animacao.html` no Chrome do computador
2. Pressione **F** para tela cheia + **H** para esconder botoes
3. Grave a tela do computador com a camera do celular (vertical!)
4. O formato ja esta otimizado para 9:16 (vertical)

### Opcao 3: OBS Studio (Melhor qualidade)
1. Baixe OBS Studio (gratuito): https://obsproject.com
2. Configure canvas para 1080x1920
3. Adicione "Captura de Janela" apontando para o Chrome
4. Grave em MP4
